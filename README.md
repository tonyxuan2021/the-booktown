# The Book Town

This web application allows the users to purchase their favoriate books.

## Stacks/Technologies

- Front-End: ReactJs, React Hooks, Context API, Sass
- Back-End: NodeJs, ExpressJs, KnexJs, MySql
- Sevices: Google Books API, Sripe
